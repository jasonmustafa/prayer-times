## Installation

### Clone

- Clone this repo to your local machine using `https://github.com/jasonmustafa/prayer-times.git`

### Setup

> install yarn and run:

```shell
$ yarn install
$ yarn start
```

> alternatively:

```shell
$ npm install
$ npm start
```
